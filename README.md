# Digital-Play 🎮

Site web professionnel de vente de produits gaming en Haïti.

## Description

Digital-Play est une plateforme e-commerce spécialisée dans les crédits pour jeux mobiles comme Free Fire, Dream League Soccer 2026 et eFootball. Le site inclut un système de wallet en HTG, authentification utilisateur, et un dashboard admin.

### Fonctionnalités
- **Page d'accueil** : Bannière gaming, produits principaux, boutons d'action.
- **Page Produits** : Catalogue détaillé avec images, prix en HTG, boutons d'achat.
- **Wallet** : Solde en HTG, recharge via MonCash/NatCash, historique.
- **Compte Utilisateur** : Inscription/connexion, dashboard avec solde, historique achats/recharges.
- **Dashboard Admin** : Gestion utilisateurs, transactions, validations recharges, ajout produits, statistiques.
- **Page Contact** : Formulaire et infos WhatsApp.

### Design
- Style gaming/e-sport moderne.
- Couleurs : Noir, Bleu néon (#00FFFF), Violet (#8A2BE2).
- Police : Orbitron (e-sport).
- Animations : Glow, hover effects.
- Responsive mobile.

### Technologies
- HTML5, CSS3, JavaScript.
- Simulation base de données avec localStorage.
- Pas de backend réel (pour démonstration).

### Comment lancer
1. Ouvrez un terminal et allez dans le dossier : `cd /workspaces/Digital-play`
2. Lancez le serveur : `python3 -m http.server 8000`
3. Ouvrez `http://localhost:8000` dans votre navigateur.

### Accès Admin
- Allez sur `http://localhost:8000/admin/admin.html` (accès simulé, pas de mot de passe).
